# Testing & Observability

Falhas de cobertura de teste e ruído excessivo de logs/observabilidade.

## Smells nesta categoria

- [Insufficient Testing](#insufficient-testing)
- [Excessive Logging](#excessive-logging)

---

## Insufficient Testing

Ocorre quando o código não tem testes adequados ou cobertura de testes insuficiente para garantir que o comportamento do sistema seja validado corretamente. Isso pode levar a erros não detectados, comportamentos inesperados e uma maior dificuldade em refatorar o código, já que não há uma rede de segurança proporcionada pelos testes automatizados.

## Excessive Logging

Ocorre quando o código possui um número excessivo de logs, dificultando a identificação de mensagens importantes. Isso pode impactar a performance, gerar arquivos de log grandes e tornar a análise de problemas mais difícil.
